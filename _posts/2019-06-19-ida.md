# [+] IDC Script

<!--more-->

사스가 아이다........... 기본적인 IDC 문법들을 정리해본다.

기본적으로 C,C++과 매우 흡사하다.

```
auto test ="shh0ya";
auto s1, s2;
s1 = test[2:4];	// 파이썬 슬라이스와 같음
s2 = test[:3];

Message("%s\n",s1);
Message("%s\n",s2);
```

```
--결과--
h0
shh
```



반복문 사용

```
auto i;

for(i=0; i<10; i++)
{
    Message("%d\n",i);
}
```


`i+=1, i+=2` 이런 형태는 안된다. 

x64에서 peb 를 찾아 IsDebugged 플래그를 바꾼다고하면...

```
auto seg;
auto Debugged;
auto peb=0;
auto tid = GetCurrentThreadId();
auto tib = sprintf("TIB[%08X]",tid);
for(seg=FirstSeg(); seg!=BADADDR; seg=NextSeg(seg))
{
    if(SegName(seg)==tib)
    {
        peb = seg;
        break;
    }
}
Debugged = peb+2;
if(Debugged!=0)
{
    PatchByte(Debugged,0);
}
```

다음과 같이 스크립트를 실행하면 패치가 아름답게 되어있다...오우우웃......
조건부 브레이크포인트도 IDC 스크립트를 이용한다고 한다.
